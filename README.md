# Coins plugin
## Description
This plugin aims to create a currency, every day minecraft players earn X coins, when they die they lose the amount configured in the plugin.
Players can give money from the command /coins pay and can not have a negative balance.

## Available storage mode

- MySQL
- SQLLite

## Server version
Tested on Paper 1.18.2

## Java version
17

## Commands
- /coins balance OR /balance => Know your account balance
- /coins pay {pseudo} OR /pay {pseudo} => Giving money to a player
- /coins baltop OR /baltop => Top richest players

## Permissions
No permission is required
